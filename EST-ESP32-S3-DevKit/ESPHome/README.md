# EST-ESP32-S3-DevKit
ESPHome example configurations for the EST-ESP32-S3-DevKit, an ESP32-S3 based development kit with Bluetooth Low Energy and native USB. [Product page](https://shop.everythingsmart.io/products/est-esp32-s3-devkit-esp32-s3-development-kit).

* [est-esp32-s3-devkit-all-gpios](est-esp32-s3-devkit-all-gpios.yaml) - helper example that exposes several common GPIOs as switches or binary sensors
* [est-esp32-s3-devkit-bluetooth-proxy](est-esp32-s3-devkit-bluetooth-proxy.yaml) - Bluetooth proxy to extend Home Assistant coverage over Wi-Fi
