# EST ESPHome Examples
Ready-to-flash ESPHome configs for Everything Smart Technology (EST) ESP32 DevKits. Shop the boards at [Everything Smart Technology](https://shop.everythingsmart.io) and use these as starting points to get firmware loaded fast via ESPHome or the included web installer (`static/index.html` / GitHub Pages: https://everythingsmarthome.github.io/firmware).

## Boards
- `EST-PoE-32` – Ethernet-first ESP32 with optional Wi-Fi fallback – [Product](https://shop.everythingsmart.io/products/est-poe-32-esp32-poe-development-board)
- `EST-ESP32-DevKit` – Standard ESP32 development board – [Product](https://shop.everythingsmart.io/products/est-esp32-devkit-esp32-development-kit)
- `EST-ESP32-C6-DevKit` – ESP32-C6 with Wi-Fi 6 – [Product](https://shop.everythingsmart.io/products/est-esp32-c6-devkit-esp32-c6-development-kit)
- `EST-ESP32-S3-DevKit` – ESP32-S3 with BLE and native USB – [Product](https://shop.everythingsmart.io/products/est-esp32-s3-devkit-esp32-s3-development-kit)

Each board folder contains example ESPHome YAMLs, and the GitHub Actions pipeline builds binaries and manifests for flashing via the web UI.
